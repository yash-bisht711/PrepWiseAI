PrepWise AI Interview Platform

PrepWise AI Interview Platform is a full-stack AI-powered interview preparation system built with the MERN stack. It simulates real interview experiences by generating role-specific questions, conducting voice-based interviews, recording candidate responses, and providing AI-driven feedback on communication, confidence, technical knowledge, and overall performance.

Features
AI-generated interview questions
Voice-based AI interviewer
Audio & video recording
Communication and confidence analysis
Technical skill evaluation
Detailed performance reports
Interview history and progress tracking
Multi-language support
Secure authentication and user profiles
Responsive modern UI
Tech Stack
Frontend: React.js, Vite, Tailwind CSS, Redux Toolkit
Backend: Node.js, Express.js, MongoDB
AI: Gemini API / OpenAI API
Authentication: JWT
Media Processing: Web Speech API, MediaRecorder API
Goal

To help students and professionals practice interviews in a realistic environment, identify weaknesses, and improve their chances of success in technical and non-technical job interviews.
